# Smart MFA Behavioural Authentication System

## Overview

Smart MFA is a Machine Learning based Multi-Factor Authentication system that combines:

- Behavioral Biometrics
- Risk Analysis
- OTP Verification using Twilio

The system analyzes user behavior such as typing speed, dwell time, flight time, touch pressure, scrolling behavior, and device tilt angle to determine whether a login attempt is legitimate or suspicious. Behavioral authentication is a recognized MFA approach that uses typing and usage patterns as an additional security factor. :contentReference[oaicite:0]{index=0}

## Authentication Workflow

1. User enters phone number, username, and password.
2. Behavioral data is collected.
3. Machine Learning model evaluates user behavior.
4. Risk score is calculated.
5. OTP is generated and sent through Twilio.
6. User enters OTP.
7. Access is granted after successful verification.
